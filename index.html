Ticket Splitter OCR

Sube un ticket, detecta artículos y reparte el pago

Aplicación local en un único HTML. No usa base de datos: los datos viven solo en memoria y se eliminan al refrescar la página. Incluye OCR con Tesseract.js, edición manual y exportación a CSV, Excel, JSON y PDF.

Sin backend · Sin base de datos · Datos temporales

Carga una imagen para activar el OCR.

👥 2. Personas
